A fully automated deployment and management system for a web application on AWS â€” built from scratch using Infrastructure as Code (Terraform) and CI/CD via GitHub Actions with OIDC. Not just a website â€” a production-grade infrastructure pipeline.

What the pipeline delivers

1) Provisions AWS Infrastructure

  • EC2 instance (Amazon Linux 2023)
  • Security Group (HTTP, SSH optional)
  • IAM Role + Instance Profile for CloudWatch Agent
  • S3 backend & DynamoDB table for Terraform state locking
  • SNS Topic + Email subscription for CloudWatch alarms
  • CloudWatch dashboard for CPU & status checks

2) Automatically deploys the site on EC2

  • GitHub Actions workflow (terraform.yml) with OIDC
  • Environment switch: dev / prod; action: apply / destroy
  • User data bootstrap script to:
    • Install Apache
    • Deploy content from app/public/
    • Show a fallback page if empty

3) Operational telemetry & alerts

  • Email alerts when instance fails (StatusCheckFailed)
  • CPU > 70% threshold notifications

Optional waiting page (wake/sleep)

  • Neon-styled S3 + CloudFront waiting page
  • Manual Wake Up button triggers Terraform Apply via API
  • Auto redirect & timed auto-sleep
